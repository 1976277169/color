Namespaces

color
Main name space. Usually contain color model classes, but also might contain other useful things. For more details see .
color::check
Test/check color
For more details see check.
color::fix
Fix color
For more details see fix.
color::place
Get index of some color component.
For more details see place.
color::operation
All operations are situated here. Some operations are not designed to be used in this form e.g. color::operation:addition
For more details see operations.
color::category
Contain all color categories. Category is pair of model and format.
For more details see category.
color::get
Get some property of color by the name. Depend of used color model this operation might be instant like getting red from RGB or require some calculation, get red from HSV model.
For more details see get.
color::set
Set some property of color. Similarly to get this operation might be instant or need calculation.
For more details see set.
color::make
Make color by given name.
For more details see make.
color::_internal
Miscellaneous internal things. Do not use entities from here. It is highly probably that this might change from version to version or from one commit to another.
