Description

Contain all function to make color according to their names.

Note

All names match names and (RGB) values specified at: http://www.w3.org/TR/css3-color

Code sample

typedef ::color::hsv< double > hsv_type;
hsv_type c;
::color::make::pink( c ); // in c is now pink color
c = ::color::make::yellow< hsv_type::category >(); // in c is now yellow color
