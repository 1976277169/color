Formats

Description

The way how specific model will be present in memory.
Depend of situation different prepresentations might be required.

See also

Memory consuptions for specific combination of model-format.

List of formats

std::uint8_t

Each componet is type of std::uint8_t.

std::uint16_t

Each componet is type of std::uint16_t.

std::uint32_t

Each componet is type of std::uint32_t.

std::uint64_t

Each componet is type of std::uint64_t.

float

Each componet is type of float.

double

Each componet is type of double.

long double

Each componet is type of long double.

color::type::split233_t

Each componet is type of std::uint8_t and everything is pack on one singe std::uint8_t. First 2 insignificant bits is used for first component, next 3 for second and last 3 for third component.

color::type::split332_t

Each componet is type of std::uint8_t and everything is pack on one singe std::uint8_t. First 3 insignificant bits is used for first component, next 3 for second and last 2 for third component.

color::type::split442_t

Each componet is type of std::uint8_t and everything is pack on one singe std::uint8_t. First 4 insignificant bits is used for first component, next 4 for second and last 2 for third component.

color::type::split556_t

Each componet is type of std::uint8_t and everything is pack on one singe std::uint16_t. First 5 insignificant bits is used for first component, next 5 for second and last 6 for third component.

color::type::split565_t

Each componet is type of std::uint8_t and everything is pack on one singe std::uint16_t. First 5 insignificant bits is used for first component, next 6 for second and last 5 for third component.

color::type::split655_t

Each componet is type of std::uint8_t and everything is pack on one singe std::uint16_t. First 6 insignificant bits is used for first component, next 5 for second and last 5 for third component.

color::type::splitAAA2_t

Each componet is type of std::uint16_t and everything is pack on one singe std::uint32_t. First 10 insignificant bits is used for first component, next 10 for second and 10 for third component and 2 for fourth.
