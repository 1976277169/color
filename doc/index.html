Color

Description

Library thats implement color.

Install

  1. Clone the project from github.
  2. Optional: Copy src/color folder in some for your suitable place.
    If you want to use directly from place where you clone color project you may skip this step.
  3. Put src/color in your path.
    If you copy src/color in someplace that is already in your path then skip this step.
Done!

API

Starting point is namespaces

Examples

In the root of the project there is folder example. It's contain one file examples e.g all examples do not need additional file to be compiled.
make_all.sh will compile all examples by using g++.
