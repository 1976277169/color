Description

Files from color library available for use or include.

List

color.hpp

Description
One file that's include all models.
Example
#include <iostream>
#include <iomanip>

#include "color/color.hpp"

int main( int argc, char *argv[] )
 {
  color::rgb< double > r;
  r = ::color::constant::lavender_type{};
  std::cout << "Red: " << r[0] << ", Green: " << r[1] << ", Blue: " << r[2] << std::endl;
  return EXIT_SUCCESS;
 }

color.pp2f.hpp

Description
All models in one file. This is pre processed file if you want to decrease number of files in your ptoject.
Example
#include <iostream>
#include <iomanip>

#define COLOR_USE_PP2FILE

#include "color/color.hpp"

int main( int argc, char *argv[] )
 {
  color::rgb< double > r;
  r = ::color::constant::lavender_type{};
  std::cout << "Red: " << r[0] << ", Green: " << r[1] << ", Blue: " << r[2] << std::endl;
  return EXIT_SUCCESS;
 }
