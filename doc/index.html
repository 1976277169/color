The _model
All color class has same set of properties like _model class.
template < typename category_name > color::_model ;
Depend of implementation color classes might or might not be in relation with _model or _model might no exist.
In here it serve purpose to describe give description of all color classes at once.
To replace all color classes in example avoid misleading that some functionality might be used only for one color class.
