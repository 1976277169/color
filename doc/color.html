Color Models

Description

Color models.

Note

All models are aliases and for type_name they use : std::uint8_t, std::uint16_t, std::uint32_t, std::uint64_t, float, double, long double, color::type::split233_t, color::type::split323_t, color::type::split442_t, color::type::split556_t, color::type::split655_t, color::type::uint24_t, color::type::uint48_t.
