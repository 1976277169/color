Namespace color::trait

Description

Every color model format combination has its own traits. Those traits can be retrieved in run or compile time.

List

bound
Description
Retrieve minimum, maximum and range of specified category.
Members
minimum
Description
Return minimum for given index.
Example
std::count << ::color::rgb< double>::bound_type::minimum() << std::endl;
maximum
Description
Return maximum for given index.
Example
std::count << ::color::hsl< double>::bound_type::maximum() << std::endl;
range
Description
Return range for given index. Equivalent to maximum()- minimum()
Example
std::count << ::color::yiq< double>::bound_type::range() << std::endl;
